## 📑 스터디 진행 방식

1. 각 장을 읽고 각자의 **감상 및 리뷰**를 `md` 파일로 작성합니다.
2. 각 폴더에 자신의 이름으로 파일을 생성하고 내용을 채웁니다.
3. 템플릿이 필요하면 `template` 폴더를 참고하세요.

## 📂 레포지토리 구조

```plaintext
dev-book-study/
├── 객체지향의 사실과 오해/

│   ├── 02. 이상한 나라의 객체/

├── template/ (템플릿 파일 포함)
└── README.md (이 파일)
```
